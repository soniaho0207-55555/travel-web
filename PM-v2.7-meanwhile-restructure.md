### Dev 操作
1. 逐条替换 `worldContext` 字符串（保留 emoji + `\n` 分隔）
2. 确认渲染层 `.timeline-world-context` 的 JS 把 `\n` 转为 `<br>` 或拆成 `<div class="mw-row">`：
   ```javascript
   // app.js 内 renderTimeline 对应位置
   const wc = node.worldContext || '';
   const html = wc.split('\n').map(row => `<div class="mw-row">${row}</div>`).join('');
   ```
3. 新增 CSS `.mw-row`（行高 1.5、行间距 4px、emoji 前 4px margin、可选把 emoji 换成 `icon('globe', 10)`）

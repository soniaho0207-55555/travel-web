# UX 眼力镜头

UX 每轮 pipeline 开工前**必读**的"小本本"。研究成果浓缩在这里，让 UX 的眼睛越来越锐利。

---

## 规则（写入前必看）

- 每条 **≤1 句话**，必须是"下次看到 X 就该亮红灯"的**可观察行为**
  - ✅ 锐：`城市页第一屏：是"标题+图"还是"打动人的句子+图"？`
  - ❌ 糊：`要有呼吸感` / `图文要匹配` / `排版要优雅`
- 每条末尾加 `→ 来源：research/<文件名>.md`
- 每段上限 **15 条**，满了要淘汰最旧或最少被激活的（淘汰记到文末"历史淘汰"）
- **只有 `/research-ux` session 能写这个文件**；`/be-ux`、pipeline 的 ux-tester subagent 只读

## 反馈 1 · 沙发时间 tab "故事 / 深一步" 穿模视觉 bug

### 【感受】❌ FAIL · 真有穿模

### 【一句话感觉】
4 tab 顶上"← back button" 直接砸在"沙发时间·读一读"文字上——像两层贴纸贴歪叠。

### 【具体到哪里】

实测 3 个穿模点：

**穿模 A · back button 压在 tab 0 文字上（最严重）**
- `.detail-back` 样式：`position: fixed; top: 20px; left: 20px; z-index: 100`
  - 矩形：(20, 20) → (52, 52)
- `.exp-beta-tab[0]` "沙发时间·读一读" 矩形：(16, -23.65) → (101.75, 48.34)
- **几何重叠区域**：(20, 20) → (52, 48) ≈ **back 按钮整个被嵌在 tab 文字范围内**
- 视觉效果：圆形 back 按钮的暗色背景 + "沙发时间·" 文字 = **像 back 按钮把"沙发时间"框成了 logo**

**穿模 B · "故事 / 深一步" 在 anchor 与 layer label 重复**
- sticky anchor `.exp-sofa-anchors`（z-index 4）链接文字：`故事` / `深一步`
- 紧接下方 `.exp-layer-label`：`故事`（20px Playfair 金沙）+ `深一步`（同款）
- **同一个词在 30-40px 距离内出现 2 次**
- 加上 tab "沙发时间·读一读" 也含"读一读"，layer-sub "沙发态·读一读" 又含"读一读"——**"读一读"在第一屏出现 3 次**

**穿模 C · 滚到第 3 section 时屏幕顶部"同代文明"标题挤到 back 按钮旁**
- 截图证据：当 anchor 切到"深一步"时，section 标题 "同代文明" 滚到 viewport top:35，而 fixed back 按钮在 top:20——两者在视觉同一行，**back 按钮的圆背景顶到"同代文明"金沙文字旁**

### 【类比锚点】
像 PowerPoint 把两个文本框拖到同一坐标 · 没人调过 z-index 的草台班子排版

### 【严重程度】**高**（页面打开第一屏即看到，CEO 看到的真实存在）

## 反馈 2 · "最大穹顶" 加粗字体明显和其他不一样

### 【感受】❌ FAIL · CEO 看到的是真的

### 【一句话感觉】
不是字体不一样——是 size + weight + color **三轴同时升级**，看起来像换字体。

### 【具体到哪里】

`<strong>最大穹顶</strong>`（在 `.exp-why-section p` 内）vs 周围 body p：

| 属性 | strong 加粗 | body p | 差异 |
|---|---|---|---|
| font-family | -apple-system, "PingFang SC" | 同 | **0**（字族相同）|
| font-size | **17px** | 15px | **+2px** |
| font-weight | **600** | 400 | **+200** |
| color | **rgb(232,212,176) 金沙 #E8D4B0** | rgb(216,207,191) 米色 #D8CFBF | **升档**（caption 米升到 hookShort 金）|
| letter-spacing | normal | normal | 0 |

**真根因**：字体没变，但**三轴同时升级**让 strong **视觉权重远超普通加粗**。
- Cereal / NYT 做加粗只升 weight（**单轴**：400→600）保持 size/color
- 当前是"升 size + 升 weight + 升 color"三轴齐升 = 杂志排版越界

### 【类比锚点】
像微信公众号加粗用了"红字加大变彩色"组合拳 · 而不是 NYT/Cereal 的"只加粗不加色"克制

### 【严重程度】**中**（每段 1-2 处 strong · 6 段累计 8+ 次穿插出现，全文充满"红字感"）

### 【出现位置】整个故事 tab "故事" layer 内：
- "是什么"段：`最大穹顶` / `两个亚伯拉罕传统都把它当过家`
- "特别在哪"段：`工程师故意让穹顶坐在光上` / `16 世纪锡南来看了一眼回去设计了蓝色清真寺`
- "深一步" 工艺卡：`掘出 Dandolo 的骨头扔给狗`
- "深一步" 争议卡：`复仇派` / `疑葬派`

## 反馈 3 · 实地看店图片歪

### 【感受】⚠️ 部分 PASS · 不是 transform rotate · 是 portrait→landscape 框架硬塞

### 【一句话感觉】
图片本身没歪，但**竖图被塞进横框**导致视觉上"东西不对劲" → CEO 口语"歪"。

### 【具体到哪里】

扫描 13 张可见 img 的 `naturalWidth × naturalHeight` vs 显示 `width × height`：

**有问题的图（natural shape ≠ frame shape）**：

| 图片 | 原图 (nw × nh) | 原图朝向 | 显示框 | 显示框朝向 | object-fit |
|---|---|---|---|---|---|
| **hagia-A3-seraphim-face.jpg** | **1200×1352** | **portrait** | 237×178 | landscape | cover |
| **hagia-A4-viking.jpg** | 1200×1200 | square | 237×178 | landscape | cover |

A3 是**撒拉弗天使脸特写**——本是竖图（脸在画面中央偏上），被塞进 237×178 的横框 + object-fit: cover：
- 浏览器只显示**中间 1200×900 区域**
- **天使的头/翅膀被裁掉**
- caption 说"主穹顶四角马赛克天使脸孔"，但图里看不到脸——**图文不符**

**preview_inspect 取值（A3）**：
- `image-orientation: from-image`（EXIF 处理 OK）
- `transform: none`（**没有 CSS rotate**）
- 真凶是 **frame aspect ratio 设错** + 原图 portrait 没有兜底

### 【类比锚点】
像把 9:16 竖屏视频塞进 16:9 电视框 · 物理没歪 · 信息歪了

### 【严重程度】**中**（A3 是"故事 tab 现场细节 carousel" 第 3 张 + 候选 B "实地看点 第 1 处 撒拉弗天使" 详情卡 = 同图被裁两次）

### 【修方建议】

3 选 1：
1. **改 frame 比例**：237×178 (4:3 横) → 237×267 (3:4 竖) · 容器变高 · 适配 portrait 原图
2. **换图源**：找一张 landscape 的撒拉弗天使图替代 A3（Wikimedia 应该有）
3. **object-fit 改 contain + bg 填充**：让原图全显示 + 底色补足（letterbox）· **不推荐**（会出黑边廉价感）

## 反馈 5 · 核心灵魂没一针见血

### 【感受】❌ FAIL · 当前 hookShort 是简历不是灵魂

### 【一句话感觉】
"做过教堂、做过清真寺、现在做第三件事"——像 LinkedIn 工作经历，不是"圣索菲亚的灵魂"。

### 【具体到哪里】

**当前 hookShort（v7.3 线上）**：
> "做过一千年教堂，五百年清真寺，现在在做第三件事。"

**第一屏 sequence**（用户先看到什么）：
1. Hero 图（圣索菲亚远景）
2. 标题 28px Playfair "圣索菲亚大教堂"
3. era 13px 金 "拜占庭·537年"
4. **hookShort 20px Playfair 金沙**（这一句决定灵魂）
5. 4 tab

**当前 hookShort 的问题**：
- "做过 / 做过 / 做" 三段排比 = **历史身份切换叙事**，不是灵魂
- "第三件事"是**悬念词**——读者会问"哪件？"，但答案藏在故事 tab 深处，第一屏没给
- **缺核心特征**：圣索菲亚的灵魂不是"它做过几件事"，而是"它是什么"

**圣索菲亚真正的灵魂候选**（CEO 提示"基督+伊斯兰融合体"开场）：

| # | 候选方向 | 示例 hookShort |
|---|---|---|
| A | **两教共用的同一座神庙** | "基督教徒拜了一千年 · 穆斯林接着拜五百年 · 同一座神庙 1500 年" |
| B | **古代世界最大穹顶 · 神浮空中** | "40 扇窗托起的穹顶 · 1500 年都让神浮在空中" |
| C | **基督+伊斯兰共享同一片光**（CEO 推荐方向）| "穹顶下 · 耶稣和真主共享同一片光 1500 年" |
| D | **第一句钩 + 第二句锚** | "全世界唯一一座 · 基督教和伊斯兰教都拜过的大教堂"|

**UX 视角推荐**：

候选 **C** 或 **A** 是"灵魂"档（独此一家的事实 + 一针见血）；
候选 B 偏"建筑奇迹"档，对教堂 vs 清真寺这件灵魂事件不够锐；
候选 D 最直白"独一无二感"，但"全世界唯一"是廉价营销词 · 风险中。

**最强**：**"基督和真主共享 1500 年同一片光"**——把"共用"+"光"+"时间"3 个锚点叠在一句。

### 【类比锚点】
当前是"履历"，灵魂应该是"独此一家的事实" · 像介绍麦当娜要说"她是一直没退场的流行女王"，不是"她出过 14 张专辑"

### 【严重程度】**高**（hookShort = 第一屏决定打开还是滑走 · 灵魂不一针见血 = 80% 用户走了）

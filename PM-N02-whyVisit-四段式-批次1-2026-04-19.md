## 字段契约回顾（写作纪律）

| 段 | chip | 字数上限 | 内容纪律 |
|---|---|---|---|
| §1 | 是什么 | ≤120 字 | 一句话定位（名字 / 时代 / 地点 / 规模 / 功能）+ 关键 meta；禁"XXX 是……"八股开头 |
| §2 | 为什么独特 | ≤150 字 | 1-2 个独占记忆点；禁"最……之一"、"著名的"这种空壳评价词 |
| §3 | 跨文明 | ≤200 字 | 至少 1 个同时期其他文明的具体锚点；禁"同期欧洲也很发达"这种泛泛 |
| §4 | 现场细节 | ≤150 字 | 至少 1 个可触达感官 / 在地锚点（颜色 / 声音 / 手感 / 具体位置 / 某件事故） |

每段末尾标注字符数（CJK + 数字 + 标点按字符数，空格不计）。

## 写作 SOP 自检（交 CEO 验收）

本批 20 段是否全部满足附录 N §N-01/§N-02 纪律？

| 纪律 | 5 景点 × 4 段 = 20 段自检 |
|---|---|
| 每段字数不超上限（120/150/200/150） | ✅ 见每段末尾字数标注 |
| §1 "是什么"禁"XXX 是……"八股开头 | ✅ 5 个 what 开头分别是：北京城 / 罗马 / 公元前 / 秘鲁 / 马丘比丘——全部以地点或时代起头，不走"某某是……"八股 |
| §2 "为什么独特"禁"最……之一"空壳词 | ✅ 没有"最""之一""著名的"等降格词 |
| §3 "跨文明"至少 1 个具体锚点 | ✅ 故宫→布鲁内莱斯基/佛罗伦萨圣母百花；斗兽场→白马寺；金字塔→龙山文化黑陶/蛋壳陶；马丘比丘→于谦景泰元年；拴日石→郭守敬《授时历》+伽利略 |
| §4 "现场细节"至少 1 个可触达感官锚点 | ✅ 故宫→金砖发乌/灰尘在朱红柱影；斗兽场→前排 4 米/活门风漏；金字塔→4500 年粉尘闷气；马丘比丘→凌晨 6:30 太阳门云海卡半山；拴日石→游客站 3 米外 |
| §3 "跨文明"是并置对照、非仅并列 | ✅ 5 段每段都有"两种方案各自回答同一个问题"或"两种相反策略"的对比结构（借鉴 Harari 笔法） |
| 斜体禁令（§L-A 硬规 1） | ✅ 无 `*斜体*` 或 `_italic_` markdown 标记；Dev 粘入时字面保持 |
| 段落内 `\n\n` 断段（§L-A 硬规 2） | ℹ️ 本批次每段单独未用 `\n\n`（每段都已 ≤200 字上限之下），Dev 粘入时字面无须断段 |

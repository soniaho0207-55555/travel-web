## Part 1 · 对照 38 条镜头 · 违反清单（按强度）

### 主要违反（修必须）

**🔵 F · 字体台阶克制（刚入库）**
- 同卡 ≥3 套字体家族：Playfair Display（拉丁 display serif）+ Noto Serif SC（CJK serif fallback）+ 系统 sans = **超 F lens 上限 ≤2 套**
- 字号 3 级：title 15px / bullets 13.5px / link 12px（其中 13.5 / 11.2 是半像素，UX-tester 实测）
- **裁定**：字体家族**已超**，字号**临界**

**🔵 字重分层 > 字号跳级**
- 当前用**字号 + 字族 + 字重 + 颜色 4 轴混用**：title (serif 15 600 #D4A574) / bullets (sans 13.5 400 #d8cfbf) / link (sans 12 default #8a7f6d)
- lens 原则："层级靠 weight + 颜色两轴"——这里多用了字号 + 字族两轴
- **裁定**：违反

**🔵 字体安静检验**
- Playfair Display 是**西文 high-contrast display serif**，盖中文走 Noto Serif SC fallback——读完会**记得**衬线感和 sans 的对比 = 字体太抢戏（反 Bear Sans "sit silently" 原则）
- **裁定**：违反

### 边缘 / 不违反

| Lens | 状态 | 备注 |
|---|---|---|
| 🔵 密度递减方向 | 边缘违反 | 卡内 title→bullets→link→bullets→title 横向密度均匀，无递减节奏 |
| 🟢 单段 viewport 上限 | 不违反 | 单段 ≤30 字 · UX-tester 实测段字数 101 |
| 🟢 图文交替呼吸 | 不违反 | 卡前后是独立 section 文字段，可承受 |
| 🟢 G · spot 卡 spotlight 层级 | 范围不同 | spot 卡场景，对照卡不直接命中 |
| 🟡 H · CJK synthetic italic 禁令 | 不违反 | 无 italic |
| 🟡 历史短文四病警报 | 不违反 | link 句"飘/埋"具象动词，不是文青腔 |

## Part 2 · "Mobile Grid fallback" 是否有 lens 覆盖？

**结论：现有 38 条 lens 没覆盖**。最接近的：

## Part 3 · lens 维度的改法（不是"我感觉"，是"应该这样"）

### 改 1 · 字体家族压回 ≤2 套（依据 F lens）

```css
/* css/styles.css:3109 */
.exp-cross-side-title {
  font-family: var(--font-sans);  /* 删 Playfair Display / Noto Serif SC */
  font-weight: 700;                /* 字重区分而非字族区分 */
  color: #E8D4B0;                  /* 金沙色 · 同 hookShort */
  font-size: 13px;                 /* 同 bullets · 字号同档 */
}
```

**效果**：同卡只剩 1 套 sans 家族 + 字重 700/400 区分 → F lens 合规

### 改 2 · 字重为主轴（依据"字重分层 > 字号跳级" lens）

- title / bullets / link **同字号 13px**（CJK body 排版三轴下限合规）
- 区分靠**字重 + 颜色 2 轴**：
  - title 700 + 金沙 #E8D4B0
  - bullets 400 + 暖灰 #d8cfbf
  - link 500 + 中灰 #8a7f6d + letter-spacing 0.06em
- 字号台阶从 3 级压到 1 级 → 上 lens "≤3 级" 严合规

### 改 3 · 桌面 right-align 取消（依据候选 I · 见 Part 4）

```css
/* 删除 css:3137-3143 */
.exp-cross-a .exp-cross-side-title { text-align: right; }   /* 删 */
.exp-cross-a .exp-cross-side-bullets { text-align: right; } /* 删 */
.exp-cross-a .exp-cross-side-bullets li::before { left: auto; right: 0; }  /* 删 */
```

aSide / bSide 都用 left-align（中文阅读习惯）。"对照感"靠**装饰条 + 字色冷暖**：
- aSide：左竖线 2px solid #D4A574（暖金）
- bSide：左竖线 2px solid #8FA5B8（冷蓝）

### 改 4 · Mobile fallback 完整重置（依据候选 I · 见 Part 4）

```css
@media (max-width: 560px) {
  .exp-cross-card {
    grid-template-columns: 1fr;
    align-items: stretch;          /* 加：不继承 desktop center */
    gap: 16px 0;                   /* 加：横 gap=0 */
  }
  .exp-cross-link {
    /* 选择 (a)：完全隐藏 hinge 文字 */
    display: none;
    /* 或 (b)：留下短分隔条 */
    /* font-size: 0; height: 1px; background: rgba(201,184,150,0.3); margin: 8px 0; */
  }
}
```

**hinge 文字"一个让神飘空中·一个让神埋岩石"**在移动端**必须不能保留**——它在桌面是横向 hinge，竖向后语义自动失效（已经在上文 whyUnique "40 扇窗让神飘在空中" 重复了"飘空中"）。

## Part 4 · 新 lens 候选（3 条 · 待 CEO 拍板下轮写入）

**候选 I · 🔵 Mobile-first responsive Grid fallback 完整性**
> Grid 容器（≥2 列）在窄屏（≤768px）fallback 必须**完整重置 5 项**：①`grid-template-columns: 1fr` ②`align-items` 不继承 desktop ③文字方向（aSide right-align → left-align）④装饰元素（hinge 横线方向/长度）⑤中央 link/hinge 语义（桌面连接句在移动端必须隐藏 / 重写 / 改装饰条）。只换 columns 不换其他 = "被压扁的桌面"，不是真 mobile-first → 来源：research/taste-CEO-对照卡-镜头诊断-2026-04-24.md

**候选 J · 🔵 西文 display 字体的 CJK fallback 风险**
> 标题用 'Playfair Display' / 'Cormorant' / 'EB Garamond' 等**西文 high-contrast display serif** + CJK fallback（Noto Serif SC / Source Han Serif）= 中文字符落衬线 fallback、与同卡 sans bullets 字体家族不一致；优先：所有 title 走单一 sans + 字重 600/700 区分；少数纪念碑式标题才用衬线 → 来源同上

**候选 K · 🟢 对照/对开页 hinge mobile 语义保留**
> 跨文明 / 跨时代对照卡桌面用"左 + 中央 link + 右"hinge 结构时，桌面 link 是连接句"一个 X · 一个 Y"；移动端单列堆叠后连接语义**自动失效**——必须 (a) 隐藏 / (b) 重写为单列适用 / (c) 改装饰条；直接保留桌面 hinge 句 = 上文已说"飘空中"的重复 + 视觉上"夹中间一行小字"莫名其妙 → 来源同上

## Part 7 · 给 CEO 的"为什么太奇怪了"白话总结

**3 个原因合起来**：

1. **同一张卡里塞了 3 套字体**：标题用衬线 Playfair（看起来像旧报刊副标题）、bullets 用系统黑体（看起来像聊天 app）、连接句用更小的灰色字 —— **眼睛要切换 3 次才读完一卡**

2. **桌面状态 aSide 文字"靠右排"**——中文阅读习惯左→右，强制让"拜占庭 · 537 年"和它下面的 bullets 都贴在中央 link 一侧的右边 = 视觉锚点反向，**像在看英文古书对开页**

3. **手机端 link 句"一个让神飘空中·一个让神埋岩石"还卡在中间**——桌面三栏并排时它是物理 hinge，手机竖向堆叠后它失去 hinge 角色 + 上文 whyUnique 已经说过"40 扇窗让神'飘在空中'" → **重复 + 莫名其妙**

**改法（白话）**：
- 字体只用 1 套（黑体），靠**字粗细**和**颜色**分层级
- 取消"靠右排"，全部左对齐
- 手机端把中间那句话**藏起来**（桌面留着可以）
- 左右对比靠**色块**或**竖线颜色**（暖金 vs 冷蓝），不靠文字方向

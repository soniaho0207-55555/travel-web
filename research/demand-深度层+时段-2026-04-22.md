### 字段建议：**升格 `visitTiming{}` · 景点级 + 城市级分层**

现 `ticket.timingTip` 是零散 string 挤在 tips 里，没结构化 = PM 写的"黄昏最好看"和"避主麻五"语义不同却挤一块。

```
景点级 visitTiming:
  bestHour: [string]     // 光线钩子："日落前 20 分钟"
  avoid:    [string]     // 硬拦："周五 13:00-14:00"
  specialDays: [string]  // 奇观："斋月夜景"

城市级 visitTiming:
  bestSeason: [string]   // "春 4-5 / 秋 10-11"（气候与城市相关，不重复在每景点）
```

`ticket.timingTip` **不删**，作为 1 行快捷摘要（从 visitTiming 自动汇总）。
